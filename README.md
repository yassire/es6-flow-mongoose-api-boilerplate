# es6-flow-mongoose-api-boilerplate
A boilerplate application for building RESTful APIs, using express and mongoose in ES6 with jest tests and JWT Authentication

Installation
------------

Clone the repository and install dependencies

```
npm install -g yarn
yarn
```

Run mongodb

```
mongod
```

Generate flow types

```
yarn flow-typed
```

Start the server

```
yarn start
```

Start in debug mode

```
yarn start:debug
```

Tests
------

Run the jest tests

```
yarn test
```

Linting
------

Run the lint

```
yarn lint
```
